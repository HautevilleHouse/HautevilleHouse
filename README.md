# HautevilleHouse

Canonical Lane is a manifold-constrained local-to-global theorem library.

This GitHub profile hosts 52 public canonical-lane repositories: reproducible theorem packages with explicit routing, artifacts, rerun protocols, and manifest-verified proof surfaces.
